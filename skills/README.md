# 🧠 AI Skills Directory

Welcome to the AI Skills directory for this project! This folder is designated for storing your custom AI instructions, prompts, automation scripts, and workflow templates.

## 📁 Suggested Structure

You can organize your files here as follows:
* `/prompts/` - For reusable prompt templates (e.g. system instructions, persona definitions).
* `/scripts/` - For automation and scripting utilities (e.g. Python, bash, node scripts).
* `/configs/` - For custom AI models or agent configurations.

## 🚀 How to Use

Place any files representing your AI tools and custom settings in this directory to keep them version-controlled along with the project.
